# JustPDF

JustPDF is just a PDF viewer for macOS.

![JustPDF](./comparison.png)

## Features

- No title bar
- No decoration
- No margin
- No ads
- No tracking
- No bullshit

## Shortcuts

- `Right Arrow` to go to the next page
- `Left Arrow` to go to the previous page
- `Cmd`+`-` to zoom out
- `Cmd`+`+` to zoom in
- `Cmd`+`0` to zoom to fit
