# JustPDF

JustPDF is just a PDF viewer for macOS.

![Comparison](./comparison.webp)

## Features

- No title bar
- No decorations
- No margins
- No ads
- No trackers
- No bullshit

## Shortcuts

- `Right Arrow` to go to the next page
- `Left Arrow` to go to the previous page
- `Cmd`+`G` to go to specific page
- `Cmd`+`-` to zoom out
- `Cmd`+`+` to zoom in
- `Cmd`+`0` to zoom to fit
